* 1、请求个人信息：
  发送 cardnumber:卡号

  ```
  {
  	"cmd_type":"require_info",
  	"content":{
        "card_number": "2198898989"
  	}
  }
  ```

  回复：

  ```
  {
  	"cmd_type":"ack_require",
  	"status_code":0,
  	"content":{
            "name": "xushun",
            "staff_number": "C1011",
            "card_number": "2198989898"
  	}
  }
  ```
  失败回复：
  ```
  {
      "cmd_type":"ack_require",
      "status_code":1,
      "content":{
            "card_number": "21989898",
            "msg": "hahahaha"
      }
  }
  ```
* 2、发送刷卡信息：

  inout:    1, -1

  ```
  {
  	"cmd_type":"refresh_time",
  	"content":{
            "card_number": "2089596016",
            "direction": 1,
            "time": "2017-10-29 13:27:00"
  	}
  }
  ```

  回复：

  ```
  {
  	"cmd_type":"ack_refresh",
  	"status_code":0,
  	"content":{
            "card_number": "21989898",
            "msg": "hahahaha"
  	}
  }
  ```
  失败回复：
    ```
  {
  	"cmd_type":"ack_refresh",
  	"status_code":1,
  	"content":{
            "card_number": "21989898",
            "msg": "hahahaha"
  	}
  }
    ```
